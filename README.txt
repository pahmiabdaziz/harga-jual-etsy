TOOLS PENENTU HARGA JUAL ETSY
=============================

Deskripsi:
----------
Aplikasi web ini membantu penjual Etsy menentukan harga jual produk secara tepat dengan mempertimbangkan berbagai biaya seperti modal, ongkir, margin, markup, fee platform, dan konversi mata uang.

Cara Menggunakan:
-----------------
1. Buka halaman web melalui link:
   https://github.com/pahmiabdaziz/harga-jual-etsy

2. Masukkan semua komponen biaya:
   - Modal pokok
   - Biaya tambahan
   - Ongkir lokal dan internasional
   - Margin & markup
   - Diskon (jika ada)
   - Biaya listing Etsy dan payment fee

3. Centang "Free Shipping" jika ongkir ingin dimasukkan ke harga jual.

4. Gunakan toggle konversi otomatis jika ingin sistem menghitung nilai konversi Etsy ke IDR (misalnya 2.5%).

5. Klik tombol **Hitung Harga Jual**

6. Salin hasil atau simpan ke local storage.

Fitur Tambahan:
---------------
- Mode Gelap / Terang (Dark Mode)
- Kurs USD realtime ke IDR (menggunakan API)
- Simpan hingga 5 riwayat terakhir di browser
- Salin hasil ke clipboard
- Visual bar margin (dengan warna indikator)
